# 「NOI2014」动物园
## Problem
[题目叙述](https://loj.ac/problem/2246)
## solve
### 分析
#### 50分算法：
用kmp求出next数组。next[i]的求出的前缀和后缀有可能重叠。怎么办？我们要缩短每个字符串的长度。我们知道，next[i], next[next[i]-1], 
next[next[next[i]-1]-1]，......都是符合要求的前缀。并且长度是递减的。在计算num[i]的时候就可以枚举这些前缀即可，同时计算答案。
#### 50分算法优化（但是多不了多少分）：
令all[i]表示以第i个字符为结尾的前缀，既是第i个前缀的前缀又是该前缀的后缀的这样的前缀有多少个。这样在求出第i个前缀的第一个符合前缀与后缀不重叠的前缀的长度之后，num[i]=all[len]（其中len表示第一个符合前缀与后缀不重叠的前缀的长度）。但是这样做依然会被"aaaaaaa"这种奇葩测试数据卡成n^2的复杂度。
#### 100分算法：
用求next数组的方法求出每一个前缀的最长的既是前缀又是后缀的字符串的长度。这样就可以优化掉求len的过程（len的定义在上面）。具体实现看代码。
关于正确性：
为了方便表述，设half[i]表示第i个前缀的最长前缀使得这个前缀也是第i个前缀的后缀。（其实就是第i个前缀len）。
half[i+1]从p = next[i]开始递归计算是不会错的。但是如果从p = half[i]开始计算呢？会不会漏掉比half[i]大的情况（因为p只有可能下降）？显然不会。因为half[i] <= i/2，half[i+1] <= (i+1)/2，在i为奇数的时候，才有可能half[i+1] = half[i] + 1。其余情况，half[i+1] <= half[i]。再说half[i+1] = half[i] + 1的情况，只有在str[i] == str[half[i]](就是 str[i] == str[p]，并且没有执行过while循环里面的东西)的时候++p，这样才会有half[i+1] = half[i] + 1。所以是这样做是正确的。
### 代码
#### 50分算法
```cpp
#include <cstdio>
#include <cstring> 

using namespace std;

const int N = 1e6 + 10, Mod = 1e9 + 7;

char str[N];
int len;

void input()
{
	scanf("%s", str);
	len = strlen(str);
}

int next[N];

void getNext()
{
	next[0] = 0;
	for (int i = 1; i < len; ++i)
	{
		int p = next[i-1];
		while (p && str[p] != str[i])
			p = next[p-1];
		if (str[p] == str[i])
			++p;
		next[i] = p;
	}
}

int num[N];

void getNum()
{
	for (int i = 0; i < len; ++i)
	{
		num[i] = 0;
		int tmp = next[i], limit = (i+1) >> 1;
		while (tmp != 0)
		{
			if (tmp <= limit)
				++num[i];
			tmp = next[tmp-1];
		}
	}
}

void output()
{
	int ans = 1;
	for (int i = 0; i < len; ++i)
		ans = (long long) ans * (num[i] + 1) % Mod;
	printf("%d\n", ans);
}

int main()
{
	int T; scanf("%d", &T);
	while (T--)
	{
		input();
		getNext();
		getNum();
		output();
	}
	return 0;
}

```

using namespace std;

		int p = next[i-1];

			p = next[p-1];
		if (str[p] == str[i])
			++p;
		next[i] = p;
	}
}

int num[N], all[N];

void getAll()
{
	for (int i = 0; i < len; ++i)
		all[i] = all[next[i] - 1] + 1;
}

{
	for (int i = 0; i < len; ++i)
	{
		int tmp = next[i], limit = (i+1) >> 1;
		while (tmp > limit)
			tmp = next[tmp-1];
		num[i] = all[tmp-1];
	}
}
